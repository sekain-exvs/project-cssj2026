# 気候変動下における主要穀物の収量変動リスクの空間的評価

機械学習ベースの気候ホットスポット分析と、2100年までの国別最適作物シフトの予測

![Python](https://img.shields.io/badge/Python-3.10-blue) ![XGBoost](https://img.shields.io/badge/XGBoost-ML-green) ![SHAP](https://img.shields.io/badge/SHAP-explainability-orange)

---

## 研究の背景と目的

気候変動は農業生産に多大な影響を与えることが予測されており、特に主要穀物であるコーン・米・小麦の収量変動は食料安全保障上の重要課題である。既存研究の多くは単一作物・単一地域の分析にとどまるが、本研究では世界規模での比較分析を行い、「2100年時点で各国にとってカロリー効率が最も高い作物は何か」という問いに対して定量的な回答を提示する。

分析には気候変数（SPEI・GSL・Hurs・TXX）と FAOstat の収量データを組み合わせた回帰データセットを構築し、勾配ブースティング系の機械学習モデル（XGBoost）で学習・予測を行う。技術固定シナリオ（農業技術が現状のまま変化しないと仮定）と、IPCC の気候シナリオ（SSP2-4.5・SSP3-7.0）を組み合わせることで、純粋な気候変動の影響のみを抽出することを意図している。

> **技術固定シナリオ**とは、農業技術・品種改良・農業政策が2013年時点の水準で固定されると仮定することで、気候変動シグナルのみを純粋に評価するための分析設計上の前提条件である。

---

## データと特徴量

回帰データセット（`regression_df_20260112_233447.xlsx`）は、FAOstat の国別年次収量データに以下の気候変数を結合したものである。前処理として国名の統一・欠損値除去・カテゴリ変数のダミー化を行い、最終的な特徴量行列を構築している。

| 変数 | 説明 |
|------|------|
| SPEI | 標準化降水蒸発散指数。乾燥・湿潤ストレスの指標。 |
| GSL | 生育シーズン長（Growing Season Length）。 |
| Hurs | 近地面相対湿度。島嶼・沿岸国で特に影響が大きい。 |
| TXX | 月最高気温の最大値。熱ストレスの代理変数。 |

これらに加え、国（Area）と穀物種別（Item）のダミー変数、および Year を特徴量として使用する。学習データは2014年以前、テストデータはそれ以降とし、将来予測では Year = 2100 として外挿する。

---

## 予測モデル

モデルには XGBoost を採用した。GridSearchCV によるハイパーパラメータ探索の結果、以下のパラメータが最適値として採用されている。外れ値除去は行わない（Model 9）。外れ値除去を行わない理由は、農業収量データにおける外れ値の多くが「実際に存在した収量変動」であり、それ自体がリスク評価の対象だからである。

```python
best_params = {
    'n_estimators': 300,
    'max_depth': 5,
    'learning_rate': 0.2,
    'subsample': 0.9,
    'colsample_bytree': 1.0,
    'random_state': 42,
    'n_jobs': -1,
}
```

モデルの解釈性を担保するため、SHAP（SHapley Additive exPlanations）による特徴量寄与分析も実施している。TreeExplainer を使用し、各気候変数・国・年が収量予測にどの程度寄与しているかをグローバル・ローカルの両視点で可視化する。

## 2100年収量予測とブートストラップ

将来の収量予測は、ISIMIP 等から取得した2096〜2099年の4年平均気候データ（SSP2-4.5・SSP3-7.0 の2シナリオ）を入力として行う。単一の予測点推定では不確実性が評価できないため、ブートストラップ法（N=100）を採用し、各試行で学習データをリサンプリングして XGBoost を再学習・予測する。これにより予測の平均値と95%信頼区間下限を全国・全穀物について出力する。

| シナリオ | 概要 |
|----------|------|
| SSP2-4.5 | 中程度の排出シナリオ。緩やかな温暖化が進む比較的楽観的なケース。 |
| SSP3-7.0 | 高排出シナリオ。現状の排出量増加が継続した場合の悲観的なケース。 |

信頼区間下限（95%CI lower）を最適作物判定の保守的基準としても使用する。これにより、不確実性が高い地域での過大評価を抑制している。

---

## 最適作物の判定方法

各国・各シナリオにおける「最適作物」は、カロリー収量（kcal/ha）が最大となる穀物として定義する。収量（kg/ha）からカロリー変換には FAO 14%水分基準のカロリー係数（Maize: 3960、Rice: 3840、Wheat: 3750 kcal/kg）を使用する。

さらに、連作障害を考慮した長期的供給係数を乗じることで単作継続時の実効収量を推定する。係数は農学文献に基づき設定されている（`continuous_cropping_supply_coefficients.md` 参照）。

| 穀物 | 連作障害補正係数 |
|------|----------------|
| Maize（コーン） | × 0.85 |
| Wheat（小麦） | × 0.80 |
| Rice（米） | × 0.88 |

対象は予測精度上位65カ国（R²基準）。2013年（2011〜2013年の3年平均）を現状ベースラインとして、2100年の各シナリオと比較することで最適作物が変化した国・変化しなかった国をチョロプレス地図で可視化する。

## 予測精度が低かった国の解釈

65カ国の選定基準から外れた91カ国については、「精度が出なかった」という事実を単純に棄却するのではなく、これらの国が一般的な回帰モデルになじまない構造的な特異性を持つ集団であることを積極的に論証する。

91カ国を観測数・収量標準偏差・気候変数の統計的特徴をもとにクラスタリングし、以下の5グループに分類している。各グループは「なぜ通常のモデルが機能しないか」を説明する独立した根拠を持つ。

| グループ | 観測数 | 収量SD | Hurs | 解釈 |
|----------|--------|--------|------|------|
| 1. 観測数少・小規模 | 50 | 1056 | 76.4 | データの希少性によりモデル学習が不十分。 |
| 2. 収量変動大・不安定 | 55 | 8358 | 64.3 | 収量SDが極めて大きく、政治・紛争等の非気候要因が支配的と推測される。 |
| 3. 島嶼・沿岸・湿度高 | 80 | 765 | 77.3 | 地形的特殊性から気候-収量の一般的な関係が成立しにくい。 |
| 4. 急成長・新興国 | 136 | 1145 | 57.2 | 農業生産性の急激な向上により、技術固定の仮定が大きく外れる。 |
| 5. 後発開発途上国（LDC） | 101 | 556 | 61.3 | 収量水準が低くデータ品質の問題が混在している可能性がある。 |
